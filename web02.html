コマ割り

  • 画面をコマでわる
  • それぞれの場所にブロックを配置する
  • 縦にならぶか、横にならぶか
  • ななめとかはちょっと...

コマ割り

        img.hoge {
          display: block;
        }
  • ブロックとは→ブロック要素(display: block;)
  • ブロックじゃない→inline
    • img
    • span
    • button
    • a

block

  • 縦にならぶ(breakされる)
  • 要素にwidth, heightがきちんとある
  • margin, paddingがある
  • 横にならべたい？→inline-block
  • デフォルトの設定はタグごとにあるが、上書きできる

inline

  • 横にならぶ(breakされない)
  • 要素にwidth, heightがきちんとしてない
  • margin, paddingがない
  • 文章の中にいれられるかどうか

おおまかに

  • コマわりはblock
  • コンテンツはinline

blockの調整

  • block間の調整はmargin
  • block内の調整はpadding

万能のflex

  • レイアウトのチート
  • いろんなレイアウトがまあまあ楽につくれる
  • 設定項目がふくざつでめんどう

中央寄せ

        div.center {
          margin: 0 auto;
        }
  • blockはmargin: auto;
  • inlineはtext-align
  • 右とか左はこれの応用
  • flexでやっちまう話もある
